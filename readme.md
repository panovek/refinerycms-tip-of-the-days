# Tip Of The Days extension for Refinery CMS.

## How to build this extension as a gem (not required)

    cd vendor/extensions/tip_of_the_days
    gem build refinerycms-tip_of_the_days.gemspec
    gem install refinerycms-tip_of_the_days.gem

    # Sign up for a https://rubygems.org/ account and publish the gem
    gem push refinerycms-tip_of_the_days.gem
